データ構造の説明
この言語学習アプリのCSVデータセットの特徴：
主要機能:

階層ナビゲーション: メインカテゴリー → サブカテゴリー → 単語 → 例文
フラッシュカード形式: 日本語⇔インドネシア語の双方向学習
ふりがな切り替え: 学習レベルに応じた表示制御
語彙展開機能: 例文内の重要語彙の詳細表示
履歴管理: 戻るボタンによるナビゲーション

エンジニア実装ポイント:

ルビ処理: {漢字|読み方}パターンを<ruby>タグに変換
階層データ構造: CSVから動的にJavaScriptオブジェクトを構築
フラッシュカード: クリックで日本語⇔翻訳を切り替え
アニメーション: CSS transitionによるスムーズな展開/収縮
状態管理: ふりがな表示状態とナビゲーション履歴

データ関係性:

words.csv ← 中心テーブル
examples.csv ← 例文（word_idで結合）
vocabulary.csv ← 語彙（example_idで結合）
subcategories.csv ← カテゴリー階層
ruby_mapping.csv ← ルビ表示制御
